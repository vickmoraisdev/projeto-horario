# Projeto Horário

Projeto desenvolvido durante os estudos de JavaScript no Curso em Vídeo.

O site identifica automaticamente o horário do sistema e altera dinamicamente a mensagem, a imagem e o estilo da página de acordo com o período do dia.

## 🔗 Acesse o projeto

🌐 [Visualizar Projeto](https://vickmoraisdev.github.io/projeto-horario/)

## 📚 Sobre o projeto

O objetivo deste projeto foi praticar a integração entre HTML, CSS e JavaScript, trabalhando principalmente com manipulação do DOM e lógica de programação.

O sistema verifica o horário atual do usuário e exibe:

- Uma mensagem personalizada
- Uma imagem correspondente ao horário
- Mudanças visuais no fundo da página

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 🎯 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

- Manipulação do DOM
- Uso de funções em JavaScript
- Estruturas condicionais (`if` e `else`)
- Trabalhar com horários usando `Date()`
- Alteração dinâmica de elementos HTML e CSS
- Organização de código
